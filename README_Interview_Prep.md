## Troubleshooting

### "I'm falling behind the timeline"
- **Solution**: Focus on your target role's critical skills
- **GenAI role**: Prioritize Parts 2-3 (LLMs, agents, RAG)
- **Traditional AI**: Prioritize Part 1 (ML/DL fundamentals)

### "The code examples don't run"
- **Check**: Python version (3.10+), package versions
- **Install**: `pip install openai langchain chromadb sentence-transformers`
- **API Keys**: Ensure OpenAI/Anthropic keys are valid

### "Questions seem too advanced"
- **Strategy**: Start with Foundational questions
- **Build up**: Master basics before Advanced questions
- **Practice**: Explain out loud to reinforce learning

### "I have interview in 2 weeks"
- **Focus**: Skim all parts, deep-dive your weak areas
- **Priority**: Part 2 (LLMs + Agents), Part 3 (System Design)
- **Practice**: Mock interviews daily
- **Portfolio**: Clean up existing projects vs building new
